# 🏋️ CoachFlow

**منصة متكاملة لإدارة الأندية الرياضية والمدربين الشخصيين**

CoachFlow هو نظام SaaS متكامل باللغة العربية يساعد المدربين الرياضيين على إدارة عملهم بالكامل — من تسجيل العملاء وتتبع الاشتراكات والمدفوعات إلى متابعة التمارين والتطورات وإصدار الخطط الغذائية والرياضية.

---

## ✨ المميزات الأساسية

### 👥 إدارة العملاء
- إضافة عملاء جدد يدوياً أو عبر استيراد Excel/CSV
- ملف شخصي لكل عميل يحتوي على: البيانات الشخصية، الهدف، الوزن، القياسات، الصور
- البحث والفلترة حسب الحالة (نشط / منتهي / معلق) والهدف
- نظام tags لتصنيف العملاء

### 📦 الباقات والاشتراكات
- إنشاء باقات تدريبية (تدريب فقط / تغذية فقط / الاثنين)
- تحديد المدة، السعر، عدد الجلسات، عدد العملاء الأقصى
- اشتراكات العملاء مع تواريخ بداية ونهاية تلقائية
- إشعارات بانتهاء الاشتراك قبل 7 أيام

### 💰 المدفوعات
- تسجيل المدفوعات (كاش / فودافون كاش / إنستاباي)
- متابعة حالة الدفع (مدفوع / غير مدفوع / جزئي)
- تقارير مالية: إجمالي الإيرادات، المبالغ المستحقة، توزيع طرق الدفع
- رسم بياني شهري للإيرادات

### 📊 متابعة التقدم
- تسجيل الوزن والقياسات (الخصر، الصدر، الذراع، الرجل، etc.)
- صور تقدم (أمامي / جانبي / خلفي / InBody)
- التزام العميل (adherence score)
- رسم بياني لتغير الوزن عبر الوقت

### ✅ متابعة الـ Check-ins
- جدولة مواعید المتابعة الدورية
- رؤية جميع العملاء المستحقين للمتابعة في لوحة واحدة
- تسجيل ملاحظات المدرب
- إشعارات للمتابعات المتأخرة

### 📋 إدارة المهام
- إنشاء مهام يومية مع تحديد الأولوية
- ربط المهام بعميل معين
- تكرار المهام (يومي / أسبوعي / شهري)
- فلترة حسب التاريخ

### 🌐 صفحة تسعير عامة
- صفحة عامة تعرض باقات المدرب للعملاء المحتملين
- رابط مباشر للاشتراك

### 📝 نظام التسجيل (Onboarding)
- نموذج تسجيل متعدد الخطوات للعملاء الجدد
- جمع البيانات الشخصية، القياسات، التفضيلات الغذائية
- رفع الصور (أمامي، جانبي، خلفي، InBody)
- تحويل تلقائي إلى lead

### 🔗 بوابة العميل
- صفحة خاصة لكل عميل (via unique token)
- تسجيل check-in أسبوعي (وزن، قياسات، صور)
- تسجيل التمارين (اسم التمرين، sets، reps، وزن)
- عرض الخطط الغذائية والرياضية المرفوعة من المدرب

### 📥 استيراد Excel ذكي
- رفع ملفات Excel/CSV
- معاينة البيانات قبل الاستيراد
- تعيين الأعمدة للحقول تلقائياً
- تحديد الباقة والتاريخ والمبلغ يدوياً
- إنشاء اشتراكات ومدفوعات تلقائياً

### 🔔 الإشعارات
- إشعارات للمدرب: انتهاء اشتراك، متابعة مستحقة، دفعة جديدة
- إشعارات للعميل: رسائل من المدرب
- علامة بعدد الإشعارات غير المقروءة

### 🛠️ لوحة التحكم (Admin)
- إدارة المدربين (إنشاء / تعليق / حذف)
- إحصائيات المنصة: عدد المدربين، العملاء، الإيرادات

### 🎨 التصميم
- واجهة عربية كاملة (RTL)
- دعم الوضع الفاتح / الداكن / تلقائي
- تصميم responsive (جوال / جهاز لوحي / مكتبي)
- رسوم بيانية متحركة

## 🛠️ التقنيات المستخدمة

| التقنية | الدور |
|---|---|
| Next.js 15 (App Router) | الإطار الرئيسي |
| TypeScript 5.8 | لغة البرمجة |
| Prisma 6 + PostgreSQL | قاعدة البيانات |
| iron-session + bcryptjs | المصادقة والتسجيل |
| Tailwind CSS 4.1 | التصميم والتنسيق |
| Radix UI | مكونات واجهة المستخدم |
| TanStack React Query | إدارة البيانات من السيرفر |
| React Hook Form + Zod | الفورمات والتحقق |
| Recharts | الرسوم البيانية |
| date-fns | التواريخ (بالتقويم العربي) |
| UploadThing | رفع الملفات |
| PapaParse + xlsx | استيراد Excel/CSV |

## 🚀 البدء

```bash
# تثبيت الاعتماديات
npm install

# نسخ المتغيرات البيئية
cp .env.example .env.local

# تشغيل قاعدة البيانات
npx prisma migrate dev

# تشغيل التطبيق
npm run dev
```
